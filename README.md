# Тестовое задание на вакансию 'Junior Python Developer'

В ходе выполнения тестового задания я предпочёл использовать Poetry — инструмент управления зависимостями. Этот выбор особенно оправдан в случае крупных проектов, где удобство и структурированность становятся важными качествами.

Для установки необходимых зависимостей выполните следующую команду:
```bash
poetry install
```

## Задание 1-3:

Для тестирования заданий используйте pytest, указывая номер задания:
```bash
pytest question-[номер задания].py
```

## Задание 4:

Для создания необходимого документа, филдов и коллекций выполните:
```bash
python question-4.py
```

## Задание 5: 

Для запуска FastAPI приложения используйте uvicorn:
```bash
uvicorn question-5:app --reload
```

После запуска обратитесь к единственному эндпоинту POST /Datalore.
